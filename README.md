# 🪑 GOLDEN MEBEL - Website

Modern, responsive furniture business website for showcasing products, portfolio, and accepting customer orders.

---

## 🚀 Live Demo
https://mebellar-plum.vercel.app/

---

## 📌 About Project

**GOLDEN MEBEL** is a premium furniture landing page designed for a furniture business in Tashkent.  
It includes portfolio, reviews, contact form, and interactive UI with modern animations.

---

## ✨ Features

- 📱 Fully responsive (mobile + desktop)
- 🎥 Hero video background
- 🍔 Mobile hamburger menu
- 🖼️ Portfolio gallery with hover effects
- ⭐ Customer reviews section
- 📩 Order/contact form UI
- 🗺️ Google Maps integration
- ⚡ Smooth animations (AOS)
- 🔥 Modern glassmorphism design
- 📞 Click-to-call buttons
- 💬 Floating quick contact buttons

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox + Grid + Glassmorphism)
- JavaScript (Vanilla JS)
- AOS Animation Library
- Font Awesome Icons
- Google Fonts (Poppins)
